## 🚀 完整使用流程

### 1. 安装依赖
```bash
cd e:\Dev\czbk\czbk-refactor
npm install
```

### 2. 开发构建
```bash
npm run dev
```
输出: `dist/lazy-sheep-auto-answer.dev.user.js`

### 3. 生产构建 (含代码混淆)
```bash
npm run build
```
输出: `dist/lazy-sheep-auto-answer.user.js`

### 4. 安装到浏览器
1. 安装 Tampermonkey
2. 打开生成的 `.user.js` 文件
3. 自动安装

### 5. 使用脚本
1. 访问 https://stu.ityxb.com
2. 点击右下角浮动按钮
3. 选择功能开始使用

## 💡 核心功能演示

### 自动答题完整流程
```javascript
// 1. 系统自动初始化
AutoAnswerSystem.init()
  ├─ 加载配置
  ├─ 注册平台适配器
  ├─ 检测当前平台
  ├─ 启动网络拦截器
  └─ 初始化UI

// 2. 用户点击"开始答题"
startAutoAnswer()
  ├─ 提取所有题目
  ├─ 云端API查询
  ├─ AI降级答题
  ├─ 填充答案
  └─ 显示进度

// 3. 自动提交
submitWork()
  ├─ 检查所有题目已答
  ├─ 点击提交按钮
  ├─ 处理确认对话框
  └─ 等待提交结果

// 4. 智能纠错 (自动)
NetworkInterceptor监听
  ├─ 捕获错题列表
  ├─ 选择纠错策略
  ├─ AI重新答题
  └─ 自动重新提交
```
