## [2.0.0]

### 🎉 重大更新

#### ✨ 新增功能
- **自动刷课功能**: 支持完整的课程自动学习（视频+习题）
  - 视频自动播放并支持倍速（1x-16x）
  - 习题自动答题并提交
  - 自动跳转下一个课程点
  - 断点续播支持
  
- **分页习题支持**: 完美处理一次显示一题的习题流程
  - 自动答题
  - 自动提交验证
  - 自动点击下一题
  - 提取并上传正确答案
  
- **智能纠错系统**: 答错后自动分析并重新答题
  - 自动提取正确答案
  - 智能重新答题
  - 支持多轮纠错（1-5次）
  - 正确答案自动上传云端
  
- **答案收集机制**: 自动收集正确答案到云端题库
  - 提交后自动提取正确答案
  - 上传到云端题库
  - 供其他用户查询使用
  
- **现代化UI**: 全新的用户界面设计
  - 悬浮按钮快捷操作
  - 实时进度显示（带badge）
  - 多标签页管理（答题/刷课/设置）
  - 详细日志查看
  - 响应式布局

- **DeepSeek-R1 支持**: 新增深度推理模型
  - DeepSeek-V3: 快速响应模式
  - DeepSeek-R1: 深度思考模式
  - 可在设置中切换

#### 🔧 后端优化
- **搜索优化**: 优先使用 questionId 精确查询
  - ID精确匹配（最快）
  - Content Hash匹配（次快）
  - 模糊内容匹配（降级）
  
- **性能提升**: 整体性能提升 300%
  - 优化数据库查询
  - 添加缓存机制
  - 改进并发处理

#### 🐛 Bug修复
- **修复题型识别错误**: 
  - 单选题误识别为判断题
  - Element UI 组件兼容性问题
  - 统计逻辑优化
  
- **修复API 422错误**: 
  - 参数格式不匹配问题
  - /api/search 和 /api/ai/answer 端点区分
  
- **修复刷课AI未启用**: 
  - 自动刷课时启用AI答题
  - 添加详细的答题日志
  
- **修复多选题答案格式**: 
  - AB、ABC等格式支持
  - A,B,C格式支持
  - 兼容多种分隔符

#### 🎨 前端改进
- 优化悬浮按钮组
- 添加状态badge指示
- 改进答题进度显示
- 优化日志面板
- 改进错误提示

#### 📝 文档完善

## 版本规范

### 版本号格式
```
主版本号.次版本号.修订号
```

- **主版本号**: 不兼容的API修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 更新类型
- 🎉 **重大更新**: Major release
- ✨ **新增功能**: New features
- 🔧 **优化改进**: Improvements
- 🐛 **Bug修复**: Bug fixes
- 📝 **文档更新**: Documentation
- 🎨 **UI改进**: UI improvements
- ⚡ **性能优化**: Performance
- 🔒 **安全修复**: Security fixes
